### My Meet App

Day 1 project skeleton for a Zoom-like meeting application.

### Stack

- **Frontend**: React, Vite, TypeScript, TailwindCSS, React Router
- **Backend**: Node.js, Express, TypeScript, PostgreSQL, Prisma (REST-style structure)
- **Shared**: TypeScript interfaces in `shared/types`

### Folder structure

- **frontend**: React SPA with basic routing and placeholder pages
- **backend**: Express server with `/api/health` route and Prisma schema
- **shared/types**: Reusable TypeScript interfaces for User, Meeting, and TranscriptSummary

### Development commands

- **Frontend dev server**: `npm run dev:client`
- **Backend dev server**: `npm run dev:server`

Run these from the project root after installing dependencies in `frontend` and `backend`.
